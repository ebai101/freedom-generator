What is this? The Freedom Generator is a Python-based sentence generator that uses Markov chains to model the statements of the GOP Presidential Candidates. It's based off of the same libraries that power similarly ridiculous projects like Subreddit Simulator.

How is it written? The generator uses the markovify Python library to create the models, and the text data is taken from The American Presidency Project's transcripts of the GOP presidential debates. You can find the data here. The source code is on Github if you're into that.
